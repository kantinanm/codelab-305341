Task 1: Configure the Router

Step 1. ที่ Toolbar ด้านบน คลิกปุ่ม Start Simulator

Step 2. จะเข้ามาที่หน้า config

Step 3. กด Enter

Step 4. พิมพ์คำสั่ง enable เพื่อเข้าสู่ Privileged Mode

enable

สัญลักษณ์หลังคำว่า Router> จะเปลี่ยนเป็นเครื่องหมาย #

Step 5. พิมพ์คำสั่ง configure terminal เพื่อเข้าสู่ Configuration Mode

configure terminal

ด้านหลัง Router จะมี (config) ซึ่งเราจะสามารถทำการตั้งค่าต่างๆ ให้ตัวอุปกรณ์ได้แล้ว

Step 6 เปลี่ยนชื่อ Router โดยพิมพ์คำสั่ง hostname ตามด้วยชื่อที่เราต้องการตั้ง ในที่นี้ให้ตั้งว่า Router1

hostname Router1

จะเห็นว่ามีการเปลี่ยนชื่อจาก Router เป็น Router1 แล้วดังรูป

Step 7. หากต้องการ Stop Simulator ให้ไปที่เมนูด้านขวามือ คลิกที่ Network Designer แล้วคลิกปุ่ม Stop Simulator ที่ Toolbar

สามารถดูวิดีโอสอนเพิ่มเติมได้ที่ https://bit.ly/2Va17UA

Task 2: Save a Configuration and then Load the Configuration

  1. หากต้องการบันทึก Configuration ให้ไปที่ Toolbar คลิกปุ่ม Save

หากมีหลาย Configured สามารถบันทึกทั้งหมดได้โดยคลิกที่ Save all configuration

  1. หากต้องการเปิดไฟล์ที่เคยบันทึกไว้ให้ไปที่ icon Open ที่ Toolbar

Task 3: Paste a Configuration

  1. หากเรามีคำสั่ง Config อยู่แล้ว และต้องการวางลงในอุปกรณ์ใน Netsim สามารถทำได้โดยสั่ง Start Simulator จากนั้นไปที่เมนู Lab > Paste Router Configuration...

  1. วางคำสั่งต่อไปนี้ลงใน Paste Configuration แล้วคลิก OK
enable
configure terminal
hostname Tampa
interface serial0/0
ip address 192.168.1.1 255.255.255.0
end

  1. พิมพ์คำสั่ง show ip interface brief
show ip interface brief

เราจะเห็นอุปกรณ์ได้มีการ Config ตามที่เราวางลงใน Paste Configuration เรียบร้อยแล้ว

เราสามารถใช้คำสั่ง 

show running-config

เพื่อดู Config ทั้งหมดได้ดังรูป
